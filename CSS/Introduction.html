            CSS - Cascading Style Sheet
            It is used to add designs to the web pages

We can add designs or styles in 3 ways / 3types;
1.Inline CSS
2.Internal CSS
3.External CSS

Inline Css:

In inline Css we use style attribute to apply CSS designs/styles to any perticular html element

NOTE:Semicolon is mandatory when we are passing more than one css property , so each and every css property should be terminated with the semicolon
But for the last css property semicolon is not mandatory

            Syntax:
                   style = "CssPropertyName : Value ;" 


            Syntax to pass multiple css properties:
                   style = "CssPropertyName:Value ; CssPropertyName:Value"

Internal Css:

In internal css we use style tag to apply CSS designs/styles to html elements

In internal css we need to select required HTML elements to apply css designs
In internal css we can select the html elements in multiple ways , here it will select all the occurences of a html element

                Syntax:
                     <style> 
                            selector{
                                CssPropertyName : Value ; 
                            }
                    </style>
                   
                
                Syntax to pass multiple css properties:
                    <style> 
                           selector{
                               CssPropertyName : Value ;
                               CssPropertyName : Value 
                           }
                   </style>

External Css:

In External css we will use external files of .css extensions and import that external file in the html file where we need to apply css designs

In External css we need to select required HTML elements to apply css designs
In External css we can select the html elements in multiple ways , here it will select all the occurences of a html element

                Syntax:
                            selector{
                                CssPropertyName : Value ; 
                            }
                   
                
                Syntax to pass multiple css properties:
                           selector{
                               CssPropertyName : Value ;
                               CssPropertyName : Value 
                           }
